# Trace Anything - Agent Guide

## Project Overview

`trace-anything` is a single-file JavaScript library that shims and traces
calls to arbitrary objects and classes. It works in both Node.js (CommonJS) and
modern browsers (no bundler required).

## Code Style

Follow the [Google JavaScript Style
Guide](https://google.github.io/styleguide/jsguide.html). The existing code in
`trace-anything.js` is the reference. Key points:

- JSDoc comments on all public methods and properties, using Closure Compiler
  type annotations
- `/** ... */` block comments for JSDoc; `//` line comments for inline notes
- 2-space indentation
- Single quotes for strings
- Semicolons required

## Environment Notes

The library targets both Node.js and browsers. The bottom of
`trace-anything.js` contains a conditional export:

```js
if (typeof module !== 'undefined') {
  module.exports = {TraceAnything};
}
```

Do not introduce `import`/`export` syntax or any runtime dependencies. The file
must remain self-contained and loadable via a plain `<script>` tag.
